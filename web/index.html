Get the Game

  1. Download or clone the repo
    git clone https://github.com/MolochDaGod/grudge-rts.git
    or use the ZIP download above.
  2. Build the Stratagus engine
    Clone Wargus/stratagus and build with CMake + Visual Studio 2022. You need stratagus.exe from the Release build.
  3. Launch
    Point the engine at the game data directory:
    stratagus.exe -d path/to/grudge-rts -p -W

Features

Sprite Pipeline

65+ Grudge Warlords characters converted to indexed PNGs with player-color palette slots for team-based rendering.

Lua Scripted

Fully data-driven via Lua scripts — units, tilesets, AI, and UI are all moddable without recompiling the engine.

Faction AI

AI crews with dynamic crew rotations, faction loyalty, and contextual behavior built on the Grudge Warlords lore.
